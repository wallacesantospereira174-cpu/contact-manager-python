# 📒 Sistema de Contatos em Python

## Sobre o projeto

Este projeto é um sistema simples de gerenciamento de contatos desenvolvido em Python.

Ele foi criado com o objetivo de praticar lógica de programação, estruturas de dados e organização de código.

Durante o desenvolvimento foram aplicados conceitos como funções, dicionários, tuplas, estruturas de repetição e tratamento de exceções.

---

## 🚀 Funcionalidades

O sistema permite:

✅ Adicionar contatos  
✅ Buscar contatos pelo nome  
✅ Listar todos os contatos cadastrados  
✅ Excluir contatos  
✅ Alterar número de telefone  
✅ Validar entradas numéricas usando `try/except`

---

## 🧠 Conceitos praticados

Neste projeto foram utilizados:

- Variáveis
- Entrada de dados com `input()`
- Estruturas condicionais (`if/elif/else`)
- Estruturas de repetição (`while` e `for`)
- Funções
- Dicionários
- Tuplas como chave de identificação
- Manipulação de dados
- Tratamento de erros com `try/except`
